== Description ==



This plugin is designed to record the video from web came and recorded video is saved on the user's youtube channel.

1) Plugin can be installed normally
2) after installation go to setting tab on wordpress admin where you will get a tag video recoditn
3) you need to set the height and width of the player and description for the page.
4) now create a page on wordpress admin panel and set the short code [videoRecording ] and publish the page
5) Now you can open the page on wordpress website and you can record the video with gmail logged in as youtube channed needs gmail logged in.
